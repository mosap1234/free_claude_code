# Modulos de Analisis Pedagogico

## 1. Taxonomias Cognitivas

### Bloom Revisado (Anderson & Krathwohl, 2001)

- Recordar, Comprender, Aplicar, Analizar, Evaluar, Crear

### SOLO (Biggs & Collis, 1982)

- Preestructural, Uniestructural, Multiestructural, Relacional, Abstracto extendido

### Webb DOK (1997)

- Nivel 1: Recuperacion
- Nivel 2: Habilidad/Concepto
- Nivel 3: Pensamiento estrategico
- Nivel 4: Pensamiento extendido

## 2. Marco Conceptual ICFES 2026

| Dimension | Valores |
|-----------|---------|
| Nivel | 1, 2, 3, 4 |
| Competencia | Interpretacion, Formulacion, Argumentacion |
| Componente | Aleatorio, Cambio, Datos, Espacial, Medida |
| Pensamiento | Numerico, Espacial, Metrico, Aleatorio, Variacional |
| Contenido | Segun area |
| Eje | Segun contexto |

## 3. Diseno de Distractores

### Tipologia de Errores

- Errores conceptuales (concepto incorrecto)
- Errores procedimentales (procedimiento incorrecto)
- Errores de calculo (operacion incorrecta)
- Errores de interpretacion (lectura incorrecta)

### Criterios de Calidad

- Plausibilidad (parece correcto)
- Diagnosticidad (revela tipo de error)
- Homogeneidad (formato similar a correcta)
- Independencia (no se eliminan entre si)

## 5. Puntuacion Compuesta

### Escala 0-100

| Rango | Calificacion |
|-------|--------------|
| 90-100 | A (Excelente) |
| 80-89 | A- (Muy bueno) |
| 70-79 | B+ (Bueno) |
| 60-69 | B (Aceptable) |
| <60 | Requiere revision |

### Componentes de Puntuacion

- Taxonomias cognitivas: 20 pts
- Validacion ICFES: 20 pts
- Distractores: 25 pts
- Principios aprendizaje: 20 pts
- Errores conceptuales: 15 pts
