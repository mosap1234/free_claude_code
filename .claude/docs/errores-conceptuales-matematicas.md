# Errores Conceptuales Comunes en Matemáticas

## 🎯 Propósito

Base de conocimiento para diseño de **distractores diagnósticos** basados en errores conceptuales reales documentados en investigación educativa.

---

## 📚 Fundamentos Teóricos

**Referencias principales**:
- Clement (1982) - Algebra word problems
- Fischbein et al. (1985) - Intuitive models
- Vinner & Hershkowitz (1980) - Concept images
- Confrey (1990) - Misconceptions vs alternative conceptions

**Principio**: Los errores NO son aleatorios. Son manifestaciones de **concepciones alternativas** sistemáticas y lógicas (desde la perspectiva del estudiante).

---

## 1️⃣ ÁLGEBRA

### Error A1: Distributividad Incorrecta de Exponentes

**Error**: (a + b)² = a² + b²

**Razonamiento erróneo**: "Si (ab)² = a²b², entonces debe aplicar igual a sumas"

**Incidencia**: 60-70% de estudiantes grados 9-10

**Origen conceptual**: Sobre-generalización de propiedad de productos

**Distractor tipo**:
```
Pregunta: (x + 3)² = ?
Correcta: x² + 6x + 9
Distractor A1: x² + 9 (error distributividad)
Distractor A2: x² + 3x + 9 (olvida duplicar término medio)
Distractor A3: 2x + 6 (confunde cuadrado con multiplicación por 2)
```

---

### Error A2: Ecuaciones con Variable en Ambos Lados

**Error**: 3x + 5 = 2x + 10 → x = 15/5 = 3

**Razonamiento erróneo**: "Sumo todo lo que está a derecha de igualdad"

**Incidencia**: 40-50% en primer contacto con ecuaciones

**Origen conceptual**: No comprender que términos con x deben agruparse

**Distractor tipo**:
```
Pregunta: 4x - 7 = 2x + 9, x = ?
Correcta: x = 8 (4x - 2x = 9 + 7)
Distractor A2: x = 1 (4x = 2x + 9 + 7, 4x = 2x + 16, error)
Distractor A2': x = 2/6 (resta incorrecta 4x-2x en numerador)
```

---

### Error A3: Simplificación de Fracciones Algebraicas

**Error**: (a + b) / a = 1 + b

**Razonamiento erróneo**: "Cancelo el 'a' arriba y abajo"

**Incidencia**: 35-45% estudiantes grados 10-11

**Origen conceptual**: Confusión entre suma y producto en numerador

**Correcto**: (a + b) / a = a/a + b/a = 1 + b/a

**Distractor tipo**:
```
Pregunta: (2x + 6) / 2 = ?
Correcta: x + 3
Distractor A3: 2x + 3 (cancela solo el denominador bajo 6)
Distractor A3': x + 6 (cancela bajo 2x incorrectamente)
```

## 2️⃣ FRACCIONES

### Error F1: Suma Incorrecta de Fracciones

**Error**: 1/2 + 1/3 = 2/5

**Razonamiento erróneo**: "Sumo numeradores y sumo denominadores"

**Incidencia**: 50-60% estudiantes grados 6-8

**Origen conceptual**: Analogía incorrecta con números naturales

**Distractor tipo**:
```
Pregunta: 1/4 + 1/6 = ?
Correcta: 5/12 (mcm = 12, 3/12 + 2/12)
Distractor F1: 2/10 = 1/5 (suma directa)
Distractor F1': 1/10 (multiplica denominadores sin ajustar)
```

---

### Error F2: División de Fracciones

**Error**: (a/b) ÷ (c/d) = (a/c) / (b/d)

**Razonamiento erróneo**: "Divido numeradores y denominadores"

**Incidencia**: 40-50% estudiantes grados 8-9

**Origen conceptual**: Generalización incorrecta desde multiplicación

**Correcto**: (a/b) ÷ (c/d) = (a/b) × (d/c) = ad/bc

**Distractor tipo**:
```
Pregunta: (2/3) ÷ (4/5) = ?
Correcta: (2/3) × (5/4) = 10/12 = 5/6
Distractor F2: 2/4 ÷ 3/5 = 1/2 × 3/5 (divide num y den)
Distractor F2': 8/15 (multiplica cruzado incorrectamente)
```

## 3️⃣ GEOMETRÍA

### Error G1: Confusión Área vs Perímetro

**Error**: Usar fórmula de perímetro para calcular área

**Razonamiento erróneo**: "Ambas miden 'cuánto tiene' la figura"

**Incidencia**: 30-40% estudiantes grados 6-8

**Origen conceptual**: No distinguir magnitudes unidimensionales vs bidimensionales

**Distractor tipo**:
```
Pregunta: Área de rectángulo 5×8 cm
Correcta: 40 cm²
Distractor G1: 26 cm (perímetro 2(5+8))
Distractor G1': 13 cm (semi-perímetro)
```

---

### Error G2: Teorema de Pitágoras Aplicado Incorrectamente

**Error**: a + b = c (en lugar de a² + b² = c²)

**Razonamiento erróneo**: "Suma directa de lados"

**Incidencia**: 25-35% en primer contacto

**Origen conceptual**: No comprender relación cuadrática

**Distractor tipo**:
```
Pregunta: Hipotenusa con catetos 3 y 4
Correcta: 5 (√(9+16) = √25)
Distractor G2: 7 (suma 3+4)
Distractor G2': 12 (multiplica 3×4)
Distractor G2': √7 (√(3+4) error)
```

---

### Error G3: Área de Triángulo sin División por 2

**Error**: A = b × h (olvida /2)

**Razonamiento erróneo**: "Igual que rectángulo"

**Incidencia**: 35-45% estudiantes grados 6-7

**Origen conceptual**: No visualizar que triángulo es "mitad" de paralelogramo

**Distractor tipo**:
```
Pregunta: Área triángulo base=6, altura=4
Correcta: 12 cm²
Distractor G3: 24 cm² (olvida /2)
Distractor G3': 10 cm² (suma b+h en lugar de producto)
```

## 4️⃣ PROBABILIDAD Y ESTADÍSTICA

### Error P1: Falacia del Jugador

**Error**: "Si salió 5 caras seguidas, ahora es MÁS probable sello"

**Razonamiento erróneo**: "Debe 'equilibrarse' pronto"

**Incidencia**: 60-70% población general (incluso adultos)

**Origen conceptual**: Confundir independencia con "compensación"

**Distractor tipo**:
```
Pregunta: Moneda justa, 4 caras seguidas. P(sello en 5to lanzamiento)?
Correcta: 1/2 (eventos independientes)
Distractor P1: 4/5 (cree que "debe equilibrar")
Distractor P1': 1/5 (cree que P(cara) aumentó)
```

---

### Error P2: Confusión P(A y B) vs P(A o B)

**Error**: P(A o B) = P(A) × P(B)

**Razonamiento erróneo**: "Combino multiplicando porque son dos eventos"

**Incidencia**: 45-55% estudiantes grados 10-11

**Origen conceptual**: No distinguir conjunción (∩) de disyunción (∪)

**Correcto**:
- P(A y B) = P(A) × P(B) [si independientes]
- P(A o B) = P(A) + P(B) - P(A y B)

**Distractor tipo**:
```
Pregunta: Dado: P(par o múltiplo 3) = ?
Correcta: P(2,4,6,3) = 4/6 = 2/3 (no contamos 6 dos veces)
Distractor P2: P(par)×P(mult3) = (3/6)×(2/6) = 1/6
Distractor P2': P(par)+P(mult3) = 3/6+2/6 = 5/6 (cuenta 6 dos veces)
```

---

### Error P3: Media vs Mediana en Distribuciones Asimétricas

**Error**: "Media siempre representa el 'centro' de los datos"

**Razonamiento erróneo**: "Promedio = valor típico"

**Incidencia**: 40-50% estudiantes grados 9-11

**Origen conceptual**: No comprender sensibilidad de media a valores extremos

**Ejemplo clásico**:
```
Datos: 1, 2, 2, 3, 3, 3, 100
Media: 16.3 (NO representa mayoría)
Mediana: 3 (sí representa valor típico)
```

**Distractor tipo**:
```
Pregunta: Salarios empresa (5 de $1M, 1 de $20M). ¿Salario típico?
Correcta: Mediana $1M
Distractor P3: Media $4.17M (influenciada por extremo)
```

## 5️⃣ PROPORCIONALIDAD

### Error PR1: Relación Inversa como Directa

**Error**: "Si 3 obreros tardan 12 días, 6 obreros tardan 24 días"

**Razonamiento erróneo**: "Más obreros → más tiempo"

**Incidencia**: 40-50% estudiantes grados 7-9

**Origen conceptual**: Sobre-generalización de proporcionalidad directa

**Correcto**: Proporcionalidad INVERSA → 6 obreros tardan 6 días

**Distractor tipo**:
```
Pregunta: 4 máquinas producen 240 piezas en 6 horas. ¿8 máquinas en cuánto?
Correcta: 3 horas (inversa: doble máquinas → mitad tiempo)
Distractor PR1: 12 horas (aplica directa erróneamente)
Distractor PR1': 6 horas (no aplica proporcionalidad)
```

---

### Error PR2: Escala Lineal en Relaciones No-lineales

**Error**: "Si lado duplica, área duplica"

**Razonamiento erróneo**: "Todo crece igual"

**Incidencia**: 50-60% estudiantes grados 8-10

**Origen conceptual**: No reconocer relaciones cuadráticas/cúbicas

**Correcto**: Si lado ×k, área ×k², volumen ×k³

**Distractor tipo**:
```
Pregunta: Círculo radio 3 cm tiene área A. Radio 6 cm tiene área...
Correcta: 4A (área ∝ r²)
Distractor PR2: 2A (relación lineal)
Distractor PR2': 8A (confunde con volumen)
```

## 6️⃣ FUNCIONES

### Error FU1: Confusión f(x+a) vs f(x)+a

**Error**: f(x+2) = f(x) + 2

**Razonamiento erróneo**: "Distribuyo el +2"

**Incidencia**: 45-55% estudiantes grados 10-11

**Origen conceptual**: No comprender función como "máquina" con input

**Ejemplo**:
```
f(x) = x²
f(x+2) = (x+2)² = x²+4x+4 ≠ x²+2
```

**Distractor tipo**:
```
Pregunta: f(x)=2x-1, f(x+3)=?
Correcta: 2(x+3)-1 = 2x+5
Distractor FU1: f(x)+3 = 2x-1+3 = 2x+2
Distractor FU1': 2x+3-1 = 2x+2 (suma 3 a x dentro de f)
```

---

### Error FU2: Pendiente como "Subida" Absoluta

**Error**: "Pendiente 2 significa sube 2 unidades"

**Razonamiento erróneo**: "Número grande = línea más vertical"

**Incidencia**: 35-45% estudiantes grados 9-10

**Origen conceptual**: No comprender pendiente como RAZÓN Δy/Δx

**Correcto**: m=2 significa Δy=2 por cada Δx=1

**Distractor tipo**:
```
Pregunta: Recta pasa por (1,3) y (4,9). Pendiente=?
Correcta: m=(9-3)/(4-1)=6/3=2
Distractor FU2: m=9-3=6 (solo Δy)
Distractor FU2': m=4-1=3 (solo Δx)
```

## 🎯 Uso en Diseño de Distractores

### Proceso Recomendado:

1. **Identificar concepto clave** del ejercicio
2. **Consultar errores comunes** de ese concepto en esta base
3. **Seleccionar 2-3 tipos de error** diferentes
4. **Calcular respuesta errónea** siguiendo razonamiento documentado
5. **Verificar plausibilidad** (¿resultado razonable?)
6. **Validar cobertura** (¿tipos de error diversos?)

### Ejemplo de Aplicación Completa:

**Ejercicio**: Área de círculo con radio 5 cm

**Respuesta correcta**: π(5)² = 25π ≈ 78.54 cm²

**Distractores basados en errores documentados**:

| Distractor | Valor | Tipo Error | Código |
|------------|-------|------------|--------|
| A | 10π ≈ 31.42 | Confunde A con C (2πr) | G1 |
| B | 25 | Olvida multiplicar por π | A3 |
| C | 5π ≈ 15.71 | Usa r en lugar de r² | PR2 |
| D | 50π ≈ 157.08 | Duplica π (error conceptual raro) | - |
| **E (✓)** | **25π ≈ 78.54** | **Correcta** | - |

**Análisis**:
- ✅ Cobertura: 3 tipos diferentes (G1, A3, PR2)
- ✅ Plausibilidad: Todos son resultados de razonamientos erróneos reales
- ✅ No obviedad: Requiere cálculo para descartar
