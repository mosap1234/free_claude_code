**Análisis detallado:**

**¿Es Interpretación y Representación?** (34% ICFES)
- [ ] Lee/interpreta gráfico, tabla, o diagrama

**¿Es Formulación y Ejecución?** (43% ICFES)
- [ ] Plantea ecuación o modelo matemático

- [ ] Calcula resultado aplicando fórmula

**¿Es Argumentación?** (23% ICFES)
- [ ] Justifica procedimiento o método
- [ ] Valida argumento o razonamiento
- [ ] Demuestra proposición matemática

- [ ] Genérico (6-9): Áreas, perímetros, Pitágoras, volúmenes básicos

## Metadatos R/exams Generados

```yaml
exname: [Nombre descriptivo del ejercicio]
extype: [schoice | cloze]
exsolution: [Respuesta correcta en formato apropiado]
exshuffle: TRUE  # FALSE en SCHOICE con opciones gráficas PNG (ver graficos-como-opciones.md)
extol: 0.01

# Metadatos ICFES (derivados del análisis anterior)
exextra[Type]: [SCHOICE | CLOZE]
exextra[Competencia]: [Resultado de Dimensión 2]
exextra[Componente]: [Resultado de Dimensión 3]
exextra[Afirmacion]: [Descripción específica basada en Dimensión 5 y 2]
exextra[Evidencia]: [Descripción específica basada en Dimensión 2 y 5]
exextra[Nivel]: [Resultado de Dimensión 1]
```

**Guía para completar metadatos:**

- **exname**: Título breve pero descriptivo (ej: "Ecuacion Lineal Contexto Velocidad")
- **extype**: "schoice" para selección única, "cloze" para pregunta compuesta
- **exsolution**:
  - SCHOICE: String binario (ej: "10000" = 5 opciones, 1ra correcta)
  - CLOZE: Según tipos (ej: "num|string|mchoice:10000")
- **exextra[Afirmacion]**: Qué sabe hacer el estudiante (desde clasificación curricular)
- **exextra[Evidencia]**: Qué evidencia específica demuestra (desde competencia)

## Errores Comunes a Evitar

### ❌ Error 1: Confundir Competencia con Componente
**Incorrecto:** Competencia: "Geométrico"
**Correcto:** Competencia: "Formulación y Ejecución", Componente: "Geométrico-Métrico"

### ❌ Error 2: Nivel por apariencia del tema
**Incorrecto:** "Tiene derivadas → Nivel 4"
**Correcto:** Contar pasos REALES del problema específico

### ❌ Error 3: Ignorar contexto en Eje Axial
**Incorrecto:** Problema de "Juan compra x manzanas" → Puramente Matemático
**Correcto:** Hay contexto narrativo → Aplicado/Contextualizado

### ❌ Error 4: Contenido sin verificar grado
**Incorrecto:** "Ecuaciones → No Genérico"
**Correcto:** Ecuaciones lineales se enseñan en grado 7-8 → Genérico

## Notas Finales

### Tips para un buen análisis

1. **Lee el enunciado completo** antes de clasificar
2. **Cuenta los pasos matemáticos** para Nivel de Dificultad
3. **Identifica la acción principal** (leer/calcular/justificar) para Competencia
4. **Pregunta "¿de qué trata?"** para Componente
5. **Pregunta "¿qué grado lo enseña?"** para Contenido
6. **Pregunta "¿hay historia?"** para Eje Axial
7. **Verifica consistencia** entre todas las dimensiones
8. **Consulta ejemplos** en caso de duda

### Flujo de trabajo recomendado

1. Leer enunciado completo
2. Completar Dimensión 1 (contar pasos)
3. Completar Dimensión 2 (acción principal)
4. Completar Dimensión 3 (área matemática)
5. Completar Dimensión 4 (tipos de pensamiento)
6. Completar Dimensión 5 (grado típico)
7. Completar Dimensión 6 (contexto)
8. Generar metadatos R/exams
9. Decidir flujo (A vs B)
10. Validar consistencia con checklist
